# Porcupine Publisher

Finance affiliate publisher stack (mobile-first, SEO-first).

## Monorepo
- `apps/web` — Astro public site
- `apps/cms` — Next.js + Payload CMS (planned)
- `packages/shared` — shared utilities + tests

## Local dev
```bash
npm install
npm run test
```
